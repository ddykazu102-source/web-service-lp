強み

01｜納期を守る進行管理

前職で工程管理を担当。スケジュール管理・タスク分解・優先順位整理を徹底します。

02｜崩れないコーディング

HTML / CSS によるレスポンシブ設計。PC・スマホ両対応で表示崩れを防ぎます。

03｜動きのある実装

JavaScriptでメニュー開閉・アコーディオン・簡単なアニメーションなどの実装にも対応可能です。

制作の流れ

  1. ヒアリング

    目的・ターゲット・参考サイト・希望納期などを確認します。

  2. 構成提案

    ワイヤー（ページ構成）を作成し、掲載内容と順番を整理します。

  3. デザイン作成

    シンプルで見やすいデザインに整えます。（参考サイトに寄せることも可能）

  4. コーディング

    HTML/CSS/JavaScriptで実装し、レスポンシブ対応も行います。

  5. 確認・修正・納品

    確認→修正を行い、完成データを納品します。
