# web-service-lp
## 概要
自分のWeb制作サービスを想定したランディングページ（LP）です。

「丁寧に、早く、崩れない。」をコンセプトに、
納期管理・情報設計・レスポンシブ対応を意識して制作しました。

---

## 制作目的
就職活動用ポートフォリオの一環として制作。
Web制作における情報整理力とUI設計力を示すことを目的としています。

---

## 使用技術
- HTML5
- CSS3
- レスポンシブ対応（メディアクエリ）
- セクション設計（情報の優先順位を意識）

---

## 工夫した点

### ① 情報構造の整理
課題提起 → 提供価値 → 強み → 流れ → 実績 → CTA  
という導線を意識し、読み手が迷わない構成にしました。

### ② 視認性を意識したデザイン
余白・文字サイズ・コントラストを調整し、
読みやすさと信頼感を重視しています。

### ③ 実務を想定した構成
実際の制作フローを明示し、
「相談〜納品」までの流れが分かるように設計しました。
